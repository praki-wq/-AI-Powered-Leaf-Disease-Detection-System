# 🌿 AI Powered Leaf Disease Detection System

## 📌 Project Overview

AI Powered Leaf Disease Detection System is an intelligent web application developed to detect and analyze plant leaf diseases using Artificial Intelligence and Computer Vision techniques. The system allows users to upload leaf images and receive disease predictions, confidence scores, symptoms, possible causes, and treatment recommendations.

The project is developed using FastAPI, Streamlit, TensorFlow/Keras, Groq AI API, and Python.

---

## 🚀 Features

- Leaf disease detection using AI
- Upload leaf images for analysis
- Disease prediction with confidence score
- Symptoms and treatment suggestions
- FastAPI backend integration
- Interactive Streamlit frontend
- Responsive user interface
- SQLite database integration
- AI-powered recommendation system

## ⚙️ Installation

### Clone Repository

git clone YOUR_GITHUB_LINK

### Create Virtual Environment

Windows:
python -m venv myenv

Mac/Linux:
python3.11 -m venv myenv

### Activate Environment

Windows:
myenv\Scripts\activate

Mac/Linux:
source myenv/bin/activate

### Install Dependencies

pip install -r requirements.txt

---

## ▶️ Run Project

### Run FastAPI

uvicorn app:app --reload

### Run Streamlit

streamlit run main.py
